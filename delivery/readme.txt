The zip file contains compiled binary files and accompanying dlls for all supported configurations and platforms.
Binary files that are part of the submission: nuacom.tsp and WebsocketAPI.dll.
All files from the single directory must be copied to the appropriate Windows System32 dir.
Also, in order for all that work on Windows 7, the following update MUST be installed:
https://support.microsoft.com/en-us/help/2999226/update-for-universal-c-runtime-in-windows
